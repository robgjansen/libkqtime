# libkqtime

a library that tracks the queuing time of bytes traveling through the kernel

## build

### out of source (recommended)

```
mkdir build
cd build
cmake .. -D<option>
make
```

### in source

```
cmake . -D<option>
make
```

### build options

 + `CMAKE_BUILD_TYPE=Debug`
 + `CMAKE_INSTALL_PREFIX=path/to/install`
